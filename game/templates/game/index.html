How to Play

I will think of a SECRET CODE (all unique digits).

You try to guess it within the allowed attempts. After each guess, I'll give you clues:

🟢
Greens: Correct digit in the CORRECT position.
🔵
Blues: Correct digit in the WRONG position.

Example:

If Code is 3 7 1 5 and you guess 3 2 5 1.

Result: 1 Green (the 3), 2 Blues (5 and 1).

🎯 Goal: Get all Greens to win!
